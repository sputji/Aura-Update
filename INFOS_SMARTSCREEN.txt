WHY DOES WINDOWS DISPLAY "WINDOWS PROTECTED YOUR COMPUTER"?

You're probably seeing a blue screen with the message "Microsoft Defender SmartScreen prevented an unrecognized application from starting."

This is perfectly normal behavior for a recent standalone application.

1. WHY?
Windows displays this warning for all .exe files that are not digitally signed with an EV (Extended Validation) certificate.
These certificates are very expensive (several hundred euros per year), which is prohibitive for free software like Aura-Update.

2. HOW DO I RUN THE SOFTWARE?

Rest assured, the software is safe. It was created by Aura Neo.

To bypass this screen:

1. Click on the "More info" link (underlined text).

2. A new button will appear: click on "Run anyway."

This operation is only necessary once. Windows will then remember that you have authorized this software.
